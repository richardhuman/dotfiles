## ZSH

* Install https://ohmyz.sh
* Clone into ~/.richard_dotfiles
* `rm ~/.zshrc`
* `ln -s ~/.richard_dotfiles/zsh/.zshrc`


## Brew

- Install Homebrew
- `~/.richard_dotfiles/brew/brew-install-base.sh`
- `brew install 1password-cli`
